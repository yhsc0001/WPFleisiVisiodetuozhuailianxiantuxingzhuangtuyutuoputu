# WPF 类似 Visio 的拖拽连线图、星状图与拓扑图

## 资源文件内容

- **拖拽连线图**：用户可以通过拖拽操作在画布上创建节点，并通过连线将节点连接起来，实现类似 Visio 的连线效果。
- **星状图**：展示了中心节点与周围节点的连接关系，适用于展示中心辐射状的拓扑结构。
- **拓扑图**：提供了基本的拓扑图绘制功能，用户可以自由绘制节点和连线，构建复杂的网络拓扑结构。
